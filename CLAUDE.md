## Project Overview
Static corporate website for Toru Inoue (井上享), a software engineer. Built with vanilla HTML, CSS, and JavaScript - no build tools or frameworks required.

## Project Structure
- `index.html` - Single-page website with sections: Hero, About, Services, Contact
- `css/style.css` - All styling
- `js/main.js` - Smooth scrolling, header effects, scroll animations using Intersection Observer

## Development
- No build process - edit HTML/CSS/JS files directly
- Open `index.html` in browser to test
- Site uses Google Fonts (Inter and Noto Sans JP)
- Bilingual navigation elements (English/Japanese)
